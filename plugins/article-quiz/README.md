# article-quiz

記事URLから5問の理解度テストを生成し、インタラクティブに出題・採点する Claude Code 用スキル。

## 仕組み

1. **記事取得** — `WebFetch` で指定URLの記事本文を抽出
2. **問題生成** — 選択式3問（4択）+ 記述式2問の計5問を生成
3. **2段階出題** — `AskUserQuestion` で選択式→記述式の順に出題（回答中は正誤非表示）
4. **採点・解説** — 全問回答後にテーブル形式で正誤・正解・解説を表示
5. **構造化出力** — 呼び出し元スキルが利用できる形式で結果を出力

## 合格基準

70%以上正解で合格（5問なら4問以上）。

## 使い方

```
/article-quiz <記事URL>
```

URLを省略した場合は `AskUserQuestion` で問い合わせる。

## 出力形式

採点完了後、以下の構造化結果を出力する:

```
## Quiz Result
- **URL**: <記事URL>
- **Score**: <正解数>/<総問題数>
- **Percentage**: <正答率>%
- **Result**: PASS または FAIL
```

呼び出し元のワークフローからこの結果を機械的に判定できる。
